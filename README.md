<h2>FormLogicLabs</h2>
<h3>form builder platform built with a high-performance monorepo architecture</h3>

<p> Features: </p>
<p> - Authentication & User Management</p>
<p> - Form Builder</p>
<p> - Different types of fields supported</p>
<p> - Public vs Private forms</p>
<p> - Response analytics dashboard</p>
<p> - Email Notification System</p>
<p> - Redis-based rate limiting</p>

<h3>Tech Stack</h3>
<p>Turborepo, tRPC, Zod, Drizzle ORM</p>
<p>Backend: Node.js, Redis, Resend </p>
<p>Frontend:Next.js, React </p>
<p>Database: PostgreSQL</p>

<h3>Installation</h3>
<p></p>Clone repository: git clone https://github.com/yourusername/formlogiclabs.git</p>

<p> Navigate into project: cd formlogiclabs</p>

<p> Install dependencies: pnpm install </p>

<h3>Create a .env file inside required apps/packages:</h3>
<p> DATABASE_URL= </p>

<p>REDIS_URL= </p>

<p>RESEND_API_KEY= </p>

<p>NEXTAUTH_SECRET= </p>

<p>NEXTAUTH_URL=</p>

<h4> Use pnpm dev and pnpm build commands to run all apps and build all apps respectively.</h4>
